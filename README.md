# 🤖 Немуся (Nemesis) — Telegram модератор-бот

**Немуся** (она же **Немесис**) — это мощный многофункциональный модератор для Telegram групп с полноценной системой рангов, варнов, мутов, банов, антиспамом на основе байесовского фильтра, проверкой ссылок и файлов через VirusTotal, премиум-подпиской через Telegram Stars, и веб-документацией со всеми командами.

Бот создан для автоматизации модерации в группах любого размера. Подходит как для небольших чатов, так и для крупных сообществ с десятками тысяч участников.

## 🚀 Основные возможности

- **Система рангов** — 5 уровней модерации (от участника до создателя)
- **Предупреждения (варны)** — с автоматическим мут/бан по достижению лимита
- **Муты и баны** — с указанием срока и причины
- **Кики** — неактивных, активных, новичков, удалённых аккаунтов, молчунов
- **Байесовский антиспам-фильтр** — самообучающийся фильтр рекламы и спама
- **VirusTotal** — проверка ссылок и файлов на вирусы (премиум-функция)
- **Триггеры** — автоматические наказания за спам, ссылки, мат и т.д.
- **Премиум** — через Telegram Stars (личный 10⭐, групповой 5⭐)
- **Инлайн-кнопки** — всё управление без команд, только нажатия
- **WebApp документация** — все 99 команд с удобным поиском и оглавлением
- **Милота** — реагирует на "Немуся" 30+ милыми фразами
- **Автоназначение создателя** — владелец группы автоматически получает ранг 5
- **Удаление сообщений** — одиночное, массовое, с тихим режимом
- **Логирование** — все действия модераторов сохраняются
- **Экспорт статистики** — выгрузка логов в CSV (премиум)
- **Автоочистка логов** — автоматическое удаление старых записей из БД

## 🛠 Технологии

- Python 3.11+
- aiogram 3.x
- SQLite (aiosqlite)
- vt-py (VirusTotal)
- Байесовский классификатор (собственная реализация)
- Telegram Stars API
