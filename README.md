# 🤖 Немуся (Nemesis) — Telegram модератор-бот

**Немуся** (она же **Немесис**) — это мощный многофункциональный модератор для Telegram групп с полноценной системой рангов, варнов, мутов, банов, антиспамом на основе байесовского фильтра, проверкой ссылок и файлов через VirusTotal, премиум-подпиской через Telegram Stars, и веб-документацией со всеми командами.

Бот создан для автоматизации модерации в группах любого размера. Подходит как для небольших чатов, так и для крупных сообществ с десятками тысяч участников.

---

## 🚀 Основные возможности

| Категория | Возможности |
|-----------|-------------|
| **Модерация** | Система рангов (0–5), варны, муты, баны, кики, предупреждения |
| **Антиспам** | Байесовский самообучающийся фильтр, капча, антифлуд |
| **Безопасность** | VirusTotal (ссылки + файлы), чёрный список слов, фильтр ссылок |
| **Автоматизация** | Триггеры (автонаказания), автокик, ночной режим |
| **Премиум** | Личный (10⭐) и групповой (5⭐) премиум через Telegram Stars |
| **Управление** | Инлайн-кнопки, WebApp документация, команда `/scan` |
| **Удаление** | Одиночное, массовое, с тихим режимом, кик неактивных/активных/новичков |
| **Логи** | Полное логирование, экспорт в CSV (премиум), автоочистка старых записей |
| **Милота** | Реакция на "Немуся" — 30+ милых фраз |

---

## 🛠 Технологии

- **Python 3.11+**
- **aiogram 3.x** — асинхронный фреймворк для Telegram Bot API
- **SQLite (aiosqlite)** — лёгкая асинхронная БД
- **vt-py** — официальный клиент VirusTotal
- **Байесовский классификатор** — собственная реализация
- **Telegram Stars API** — приём платежей через звёзды

## 📦 Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/Narek-D8v/Nemesis-bot.git
cd Nemesis-bot
2. Установить зависимости
bash
pip install -r requirements.txt
3. Настроить окружение
bash
cp .env.example .env
Отредактируйте .env и укажите:

text
BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_ID=ваш_Telegram_ID
VIRUSTOTAL_API_KEY=ваш_API_ключ_VirusTotal
4. Запустить бота
bash
python main.py
